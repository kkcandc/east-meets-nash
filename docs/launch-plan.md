# Launch Plan

Target public launch: Thursday, May 14, 2026.

## Launch Goal

Have a real-looking, content-rich East Nashville site live with email signup, a daily brief, social handles, source-backed posts, sponsor offer, and enough attitude that people understand it immediately.

## Before Launch

Day 1:

- Finalize name, domain, handles, and email identity.
- Stand up web app scaffold.
- Create beehiiv publication and signup embed/API path.
- Create social accounts.
- Create first reporter headshots and voice prompts.

Day 2:

- Build source ingestion list.
- Pull first source batch.
- Create 20 to 40 starter stories or briefs.
- Build homepage, article template, sponsor page, and search/category pages.

Day 3:

- Create daily brief template.
- Create weekly issue template.
- Publish first web posts.
- Build ATXP login proof of concept if credentials are available.

Day 4:

- Start social posting.
- Create sponsor self-serve placeholder.
- Add analytics.
- Create correction/takedown flow.

Day 5:

- Quality pass on tone, mobile UI, SEO, source links, and article density.
- Publish launch morning brief.
- Seed comments/reactions only if clearly marked or generated from real early users.

## Launch Content Mix

- 1 main feature.
- 8 to 12 quick hits.
- 5 restaurant/bar items.
- 5 events.
- 3 civic/watchdog items.
- 3 "what people are mad about" items.
- 1 classifieds starter page.
- 1 sponsor offer page.

## Launch Metrics

- Subscribers.
- Site visits.
- Story shares.
- Reaction rate.
- Comment rate.
- Sponsor page visits.
- Sponsor purchases/leads.
- Tip emails.
- Search impressions once indexed.

## Immediate Post-Launch

- Publish daily.
- Watch what people forward.
- Build more of the beats that get reactions.
- Start pitching founding sponsor package through the self-serve page and direct links.
- Keep the weekly issue more curated than the daily stream.
